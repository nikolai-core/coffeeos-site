What's inside

More than a kernel.
A full stack.

Aurora Refresh ships a real desktop, real apps, real persistence, and real hardware drivers — freestanding, -nostdlib, no shortcuts.

Higher-half Kernel

GDT, TSS, IDT, paging, PMM and VMM layers. GRUB2 Multiboot2 entry. Runs entirely above the 3 GB virtual mark.

Framebuffer Desktop

Movable windows, icons, taskbar, and app launcher over a linear framebuffer compositor with double-buffering.

Built-in App Suite

Notepad (Ctrl+S/N/O), Paint, Calculator, Clock, Mixer, System Info, Files — all running natively in coffeeOS.

FAT32 + VFS

Full FAT32 read/write with sector cache, MBR parsing, VFS layer, ATA PIO disk driver, and ramdisk boot image.

RTL8139 Network

RTL8139 NIC driver with QEMU user networking. Attached automatically on just run-persist targets.

SB16 Audio

Sound Blaster 16 DMA audio path with automatic PC speaker fallback. Controlled via Mixer app and syscall interface.

Kernel Shell

Shared dispatcher for the kernel shell and GUI terminal: ls, cat, write, mkdir, rm, cp, mv, stat, df, touch, edit, sync.

Syscall Plumbing

Userland syscall dispatch wired through the kernel. Covers filesystem, I/O, and process-level operations.

Get running

Build & run

Powered by a justfile. Requires a i686-elf-gcc cross-compiler.

essential commands
$just rebuild  # clean full rebuild
$just run-persist  # ATA + RTL8139
$just run-persist-audio  # + SB16
$just mkdisk  # new persist.img
$just clean  # remove build outputs
windows vhd workflow
$just img-to-vhd
# persist.img → persist.vhd
# edit in Windows Disk Management
$just vhd-to-img
# persist.vhd → persist.img
⚠ don't mount VHD while QEMU runs
requirements
•gcc  # i686-elf cross-compiler
•ld
•grub-mkrescue
•qemu-system-i386
•python3
•just  # task runner
persistence notes
build/disk.img
└─ FAT32 ramdisk in ISO
build/persist.img
└─ MBR + FAT32, ATA attached
$sync  # flush before exit
$reboot  # or reboot to flush

Roadmap

What's next

Aurora Refresh is the most complete version yet. The brew continues.

v1.0 — done

Boot + Interrupts + VGA

GRUB2 Multiboot, GDT, IDT, VGA text, PS/2 keyboard, PIT.

v1.1 — done

Paging + Memory Mgmt

Higher-half mapping, PMM bitmap, VMM, heap allocator, TSS.

v1.2 — done

Framebuffer Desktop + FAT32

Linear framebuffer, windowing system, VFS, FAT32 driver, ATA PIO.

v1.3 — current — Aurora Refresh

Apps, Audio, Networking & Syscalls

Full app suite, SB16 audio, RTL8139, userland syscall dispatch, Windows VHD workflow, serial logging, mouse support.

v1.4 — planned

Preemptive Scheduler

True multitasking with context switching, TCBs, round-robin priority queue.

v2.0 — future

ELF Loader + TCP/IP

Load ELF userspace binaries, minimal TCP/IP over RTL8139, ext2 read support.
